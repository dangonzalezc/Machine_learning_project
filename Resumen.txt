================================================================================
    PROYECTO: MACHINE LEARNING PARA CLASIFICACION DE ADENOCARCINOMA
================================================================================

OBJETIVO:
---------
Desarrollar modelos de Machine Learning para clasificar pacientes con 
adenocarcinoma vs sanos utilizando biomarcadores y variables clinicas.

DATASET SIMULADO:
-----------------
    Total de pacientes: 500
    - Sanos: 250 (50%)
    - Adenocarcinoma: 250 (50%)

VARIABLES:
----------
    [CONTINUAS - BIOMARCADORES]
    1. CEA (Antigeno Carcinoembrionario)
       - Sanos: ~2.5 ng/mL
       - Cancer: ~25 ng/mL
    
    2. CA19-9 (Marcador tumoral)
       - Sanos: ~15 U/mL
       - Cancer: ~120 U/mL
    
    3. CA125 (Marcador tumoral)
       - Sanos: ~20 U/mL
       - Cancer: ~85 U/mL
    
    4. AFP (Alfa-fetoproteina)
       - Sanos: ~5 ng/mL
       - Cancer: ~30 ng/mL
    
    5-7. GeneExpr1, GeneExpr2, GeneExpr3
       - Expresion genica diferencial

    [CATEGORICAS - VARIABLES CLINICAS]
    1. Edad: <40, 40-60, >60 anos
    2. Sexo: M, F
    3. Fumador: No, Ex-fumador, Fumador
    4. Antecedentes Familiares: Si, No
    5. Estadio: I, II, III, IV (solo cancer)

MODELOS IMPLEMENTADOS:
----------------------
    1. Regresion Logistica
       - Modelo lineal interpretable
       - Baseline de comparacion
    
    2. Random Forest
       - Ensemble de arboles de decision
       - Alta precision y robustez
    
    3. Gradient Boosting
       - Boosting secuencial
       - Excelente desempeno
    
    4. Support Vector Machine (SVM)
       - Kernel RBF
       - Clasificacion no lineal

METRICAS DE EVALUACION:
-----------------------
    - Accuracy: Precision general
    - Precision: VP / (VP + FP)
    - Recall (Sensibilidad): VP / (VP + FN)
    - F1-Score: Media armonica de Precision y Recall
    - AUC-ROC: Area bajo la curva ROC
    - Matriz de Confusion: Analisis detallado

VISUALIZACIONES INCLUIDAS:
--------------------------
    [ANALISIS EXPLORATORIO]
    + Histogramas de distribucion (7 biomarcadores)
    + Box plots comparativos (Sano vs Cancer)
    + Matriz de correlacion (heatmap)
    + Pair plots (relaciones bivariadas)
    + Graficas de barras (variables categoricas)
    + Pie chart (distribucion de estadios)
    
    [EVALUACION DE MODELOS]
    + Grafica comparativa de metricas (barras agrupadas)
    + Matrices de confusion (4 modelos)
    + Curvas ROC (todas en una)
    + Curvas Precision-Recall
    + Importancia de caracteristicas (Random Forest)
    + Coeficientes (Regresion Logistica)

RESULTADOS ESPERADOS:
---------------------
    > AUC-ROC: 0.90 - 0.98 (excelente discriminacion)
    > Accuracy: 85% - 95%
    > Precision: 85% - 95%
    > Recall: 85% - 95%

CARACTERISTICAS MAS IMPORTANTES:
---------------------------------
    1. CA19-9 (marcador tumoral mas discriminativo)
    2. CEA (antigeno carcinoembrionario)
    3. CA125 (marcador tumoral)
    4. GeneExpr1 (sobreexpresion en cancer)
    5. Edad (mayor prevalencia en >60 anos)
