┌────────────────────────────────────────────────────────────────────────────┐
│ PASO 1: INSTALAR DEPENDENCIAS                                             │
└────────────────────────────────────────────────────────────────────────────┘

Abre una terminal (CMD o PowerShell) en esta carpeta y ejecuta:

    pip install -r requirements.txt

Esto instalara:
    ✓ numpy
    ✓ pandas
    ✓ matplotlib
    ✓ seaborn
    ✓ scikit-learn
    ✓ jupyter


┌────────────────────────────────────────────────────────────────────────────┐
│ PASO 2: INICIAR JUPYTER NOTEBOOK                                          │
└────────────────────────────────────────────────────────────────────────────┘

En la misma terminal, ejecuta:

    jupyter notebook

Esto abrira Jupyter en tu navegador (normalmente en http://localhost:8888)


┌────────────────────────────────────────────────────────────────────────────┐
│ PASO 3: ABRIR EL NOTEBOOK                                                 │
└────────────────────────────────────────────────────────────────────────────┘

En la interfaz web de Jupyter:

    1. Busca el archivo: adenocarcinoma_ml_analysis.ipynb
    2. Haz clic en el para abrirlo


┌────────────────────────────────────────────────────────────────────────────┐
│ PASO 4: EJECUTAR EL ANALISIS                                              │
└────────────────────────────────────────────────────────────────────────────┘

OPCION A - Ejecutar todo de una vez:
    Menu superior: Cell → Run All
    
OPCION B - Ejecutar celda por celda:
    Presiona: Shift + Enter en cada celda
    
OPCION C - Ejecutar hasta cierto punto:
    Menu superior: Cell → Run All Above


┌────────────────────────────────────────────────────────────────────────────┐
│ QUE VERAS AL EJECUTAR                                                      │
└────────────────────────────────────────────────────────────────────────────┘

El notebook ejecutara automaticamente:

    [✓] Generacion de 500 pacientes simulados
    [✓] Analisis exploratorio con ~20 graficas
    [✓] Entrenamiento de 4 modelos ML
    [✓] Evaluacion comparativa
    [✓] Visualizaciones avanzadas
    [✓] Reporte de metricas y conclusiones

TIEMPO TOTAL: ~2-3 minutos


┌────────────────────────────────────────────────────────────────────────────┐
│ ESTRUCTURA DEL NOTEBOOK                                                    │
└────────────────────────────────────────────────────────────────────────────┘

    Seccion 1: Generacion de Datos Simulados
        → Crea dataset de 500 pacientes
        → 7 biomarcadores + 4 variables clinicas
    
    Seccion 2: Analisis Exploratorio (EDA)
        → Histogramas de distribucion
        → Box plots comparativos
        → Matriz de correlacion
        → Pair plots
        → Analisis de variables categoricas
        → Distribucion de estadios
    
    Seccion 3: Preparacion de Datos
        → Division train/test (75/25)
        → Normalizacion de caracteristicas
        → Codificacion de variables categoricas
    
    Seccion 4: Modelos de Machine Learning
        → Regresion Logistica
        → Random Forest
        → Gradient Boosting
        → Support Vector Machine (SVM)
    
    Seccion 5: Comparacion de Modelos
        → Tabla de metricas
        → Matrices de confusion
        → Curvas ROC
        → Curvas Precision-Recall
    
    Seccion 6: Importancia de Caracteristicas
        → Ranking de biomarcadores
        → Coeficientes de modelos
    
    Seccion 7: Conclusiones
        → Resumen de resultados
        → Recomendaciones


┌────────────────────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────────────────────┘

      Haz clic derecho → Abrir imagen en nueva pestana
    
    • Para modificar parametros:
      Edita las celdas de codigo y vuelve a ejecutar
    
    • Para guardar tu trabajo:
      File → Save and Checkpoint

┌────────────────────────────────────────────────────────────────────────────┐
│ SOLUCION DE PROBLEMAS                                                      │
└────────────────────────────────────────────────────────────────────────────┘

PROBLEMA: "ModuleNotFoundError: No module named 'xxx'"
SOLUCION: pip install xxx

PROBLEMA: "Kernel appears to have died"
SOLUCION: Kernel → Restart → Run All

PROBLEMA: Las graficas no se muestran
SOLUCION: Verifica que la celda tenga: %matplotlib inline

PROBLEMA: "NameError: name 'df' is not defined"
SOLUCION: Ejecuta las celdas anteriores primero

PROBLEMA: Jupyter no abre en el navegador
SOLUCION: Copia la URL que aparece en la terminal
